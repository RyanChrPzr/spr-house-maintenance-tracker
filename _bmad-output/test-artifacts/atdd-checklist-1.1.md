# ATDD Checklist — Epic 1, Story 1.1: Property Setup Onboarding Flow

**Date:** 2026-03-04  |  **Author:** HackathonTeam2  |  **Level:** Unit + Integration

---

## Story Summary

As a homeowner, I want to set up my property (type and optional age) so the app can generate
a relevant maintenance schedule. First-time homeowners are redirected to property setup before
seeing the dashboard. On save, a `properties` row is created and `maintenance_tasks` is seeded.

---

## Acceptance Criteria

1. **AC1** — First-time homeowner reaching `/homeowner/dashboard` is redirected to property setup screen
2. **AC2** — Select property type + (optional) age + save → `properties` row created → task template selection screen
3. **AC3** — Saving without selecting property type → validation error "Please select a property type."
4. **AC4** — On save, `maintenance_tasks` table is pre-populated with default tasks for the property type
