# ATDD Checklist — Epic 1, Story 1.3: Recurring Schedule Configuration & Task Customisation

**Date:** 2026-03-04  |  **Author:** HackathonTeam2  |  **Level:** Unit + Integration

---

## Story Summary

As a homeowner, I want to set recurrence intervals and customise individual tasks so my
maintenance schedule matches my home's actual needs. Changing the interval recalculates the
next due date immediately. Custom names and notes persist to Supabase.

---

## Acceptance Criteria

1. **AC1** — Tapping a task card opens detail/edit screen with: task name, interval dropdown, notes, next due date
2. **AC2** — Changing interval + save → interval updated + next due date recalculated immediately
3. **AC3** — Updating name or notes + save → reflected in task list immediately
4. **AC4** — "Add Task" button opens blank creation form
