# ATDD Checklist — Epic 1, Story 1.2: Pre-loaded Maintenance Task Template Selection

**Date:** 2026-03-04  |  **Author:** HackathonTeam2  |  **Level:** Unit + Integration

---

## Story Summary

As a homeowner who completed property setup, I want to pick from 6 pre-loaded Filipino home
maintenance task templates so I don't have to build my schedule from scratch. Each selected
task is created in `maintenance_tasks` with its default recurrence interval.

---

## Acceptance Criteria

1. **AC1** — Template screen shows at minimum: Aircon Cleaning, Pest Control, Plumbing Check, Septic Tank Pump-out, Rooftop Inspection, Electrical Check
2. **AC2** — Tapping a template selects/deselects it
3. **AC3** — "Confirm Selection" → each selected task created with default interval → navigate to `/homeowner/tasks`
4. **AC4** — Confirming with no selection → prompt "Select at least one task to activate your schedule."

---
